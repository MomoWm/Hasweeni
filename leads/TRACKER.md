## Status options
- **Not started** — haven't reached out yet
- **Demo sent** — sent the free demo + first message
- **Followed up** — sent the one allowed follow-up
- **Responded** — they replied, in conversation
- **Closed — paid** 🎉 — they paid, site published
- **Passed** — not interested, move on

## How to add a new lead
1. Find it via Google Maps (see the "finding more leads" section in any file under `leads/`)
2. Add a new row here
3. Add it to the right niche file (or create a new one, e.g. `leads/[city]-[niche]-leads.md`)
