# Plumbing Business Website Template

A clean, modern, mobile-friendly website template for pitching plumbing
businesses that don't have a website yet. Pure HTML/CSS/JS — no frameworks,
no build tools, no dependencies, no hosting costs.

## Files
- `index.html` — the page content and structure
- `style.css` — all styling (colors are variables at the top — easy re-skin)
- `script.js` — mobile menu + small interactions
- `CUSTOMIZE.md` — step-by-step guide to personalize this for a specific business

## This template is already live
This folder lives in the shared `Hasweeni` GitHub repo, which already has
GitHub Pages enabled. Once customized and pushed, any plumbing demo built
from this template is reachable at:
`https://momowm.github.io/Hasweeni/<folder-name>/`

## Building a new client demo from this template
1. Copy this whole folder to a new folder named after the business (e.g. `fix-it-right-plumbing/`)
2. Follow `CUSTOMIZE.md` to fill in the business's real name, phone, city, and colors
3. Commit and push — it's live within about a minute

See the top-level [`MASTER-GUIDE.md`](../MASTER-GUIDE.md) for the full outreach process this template fits into.
