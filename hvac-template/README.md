# HVAC Business Website Template

A clean, modern, mobile-friendly website template built for pitching HVAC
(heating & air conditioning) businesses that don't have a website yet. Pure
HTML/CSS/JS — no frameworks, no build tools, no dependencies, no hosting
costs.

## Files
- `index.html` — the page content and structure
- `style.css` — all styling (colors are variables at the top — easy re-skin)
- `script.js` — mobile menu + small interactions
- `CUSTOMIZE.md` — step-by-step guide to personalize this for a specific business

## Quick start
1. Open `CUSTOMIZE.md` and follow the steps to swap in the client's business
   name, phone number, city, and colors.
2. Open `index.html` directly in a browser to preview it locally — just
   double-click the file, no server needed.
3. When it looks good, publish it for free (see below).

## Publish for free — GitHub Pages (recommended, $0 forever)

1. Create a free GitHub account at [github.com](https://github.com) if you don't have one.
2. Create a new repository (e.g. `johnson-hvac-site`).
3. Upload these three files (`index.html`, `style.css`, `script.js`) to the repo.
4. Go to the repo's **Settings → Pages**.
5. Under "Source," select the `main` branch and save.
6. Wait ~1 minute — GitHub gives you a live link like:
   `https://yourusername.github.io/johnson-hvac-site/`

That link is a real, working website you can send to the business owner —
no cost, no expiration, no credit card ever required.

## Reusing this for multiple clients

Keep this folder as your "master template." For each new prospect:
1. Copy the whole folder to a new folder (e.g. `plumbing-template` won't
   apply here, but `smith-hvac`, `abc-heating-air`, etc.)
2. Follow `CUSTOMIZE.md` for that business
3. Push it to its own new GitHub repo and publish

This is exactly how you build a demo site fast for cold outreach — a
finished, professional-looking site with the business's own name on it beats
any cold pitch.
